**Los clientes confían en un guía que tenga un
plan.**

Realizar una compra es un gran paso.
Por eso querrán un camino claro que le
saque hasta la ultima duda de hacer
negocios con nosotros.


**Los clientes confían en un guía que tenga un
plan.**

El hecho de saber que queremos que
hagan un pedido no basta para
motivarlos.


**Plan de proceso**

Este plan debe mostrarle simplificadamente al
cliente como hace para adquirir el producto o
servicio y/o como usarlo una vez comprado.
Es decir, tendremos un plan preventa y un plan
posventa.

Al leer tu plan debemos sacarle un:

```
“Ay, no es difícil, puedo hacerlo”
```

**Plan de proceso**

Los pasos deben ir entre 3 a 6 pasos. Cuanto
menos mejor. Lo podés llamar, por ejemplo: Plan
de instalación fácil.

**Recordar que invadir al cliente con mucha
información es inversamente proporcional a la
venta.**


**Plan de acuerdo**

Un plan de acuerdo sirve para palear el miedo.
Vendría a ser como el telón de fondo. Cuando mas
nos vamos conociendo con el cliente, este buscará
indagar cada vez mas, hasta llegar al plan de
acuerdo.

Por ejemplo, lo puedes denominar: Acuerdo de
satisfacción del cliente


**Llamado a la acción**

Los clientes no actúan salvo si se les
desafía para que actúen.
Un llamamiento para actuar es directo:
se le pide al cliente que compre o que
concierte una cita.
**Si no pedimos la compra, el cliente
puede percibirlo como debilidad**


**Dos tipos de llamamiento a actuar**

1) Llamamiento directo para actuar:
Comprar ahora, concierta una cita,
suscríbete hoy.
2) Llamamientos transicionales para
actuar: Ofrecer algo al cliente de
manera gratuita (un pdf). Invitarlo
a un webinario.


**Llamamiento transicional para actuar**

Tiene 3 funciones muy potentes:
Establecer tu territorio: si quieres ocupar
ese territorio, reclámalo!!
Crear reciprocidad: Cuanto mas generosa
sea una marca mas reciprocidad genera.
Posicionarte como guía: Al solucionar un
problema te estás posicionando como
guía.


**Ideas para llamamiento transicional para actuar**

Ejemplos:
Información gratuita: Un capítulo de libro.
Testimonios: Video o pdf. Mostrando los clientes
felices.
Muestras: Muestras gratuitas del producto.
Período de prueba gratuito: Dar días de
“prueba”. Si el producto es muy bueno, la
conversión será alta.

# Repositorios para DVR/Camera Server en Android

## Lo que describes suena a **IP Webcam** o un servidor de cámara en Android

### Repositorios Open Source más conocidos:

## 1. **DroidCam** (Open Source)
```
https://github.com/dev47apps/droidcam
```
- Convierte Android en webcam/IP camera
- Tiene cliente para PC

## 2. **IP Webcam (Pavel Khlebovich)**
- La app más popular pero **no es open source**

## 3. **SpyDroid / iSpy**
```
https://github.com/esimov/android-spydroid
```

## 4. **AndroidCamera RTSP Server**
```
https://github.com/pedroSG94/RootEncoder
```
- Transmite RTSP/RTMP desde Android
- **Muy similar al camera_server del ESP32**
- Permite modificar completamente

## 5. **LibStreamingEngine**
```
https://github.com/fyhertz/libstreaming
```

---

## ¿Cuál se parece MÁS al ESP32 Camera Server?

**RootEncoder** de pedroSG94 porque:
- Expone stream por red local
- Soporta RTSP
- Es modificable desde Android Studio

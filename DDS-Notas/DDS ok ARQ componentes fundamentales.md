Componentes fundamentales de soporte en Arquitectura y sus funciones principales son:
	 - Administración: 
		 - Coloca el servicio en los nodos 
		 - identifica errores
		 - reequilibra los servicios entre nodos
	 - Detección de Servicios
		 - mantiene la lista de servicios y los nodos donde se encuentran.
	 - API Gateway . Puerta de Enlace API
		 - Es quien va redirigir distintos servicios , los clientes llaman directamente a este componente . Agrega respuesta de varios servicios, también funciona como autenticación, el registro, la terminación SSL y el equilibrio de carga.
